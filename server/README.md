# MERN CRUD - Server

## Setup
1. Copy `.env.example` to `.env` and set `MONGO_URI` if needed.
2. Install dependencies:
   ```
   cd server
   npm install
   ```
3. Run server:
   ```
   npm run dev
   ```
4. API endpoints:
   - GET /api/users
   - POST /api/users
   - PUT /api/users/:id
   - DELETE /api/users/:id
